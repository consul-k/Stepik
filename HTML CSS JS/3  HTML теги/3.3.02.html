Рецепт блюд:
  1. Домашние сырные палочки
    1. Ингредиенты
      • Сыр твердый 300 г
      • Яйцо 1 шт
      • Мука 30 г
      • Масло растительное 70 мл
    2. Приготовление
      • Сыр натереть на мелкой терке
      • Добавить муку, яйцо. Перемешать
      • Сформировать сырные палочки
      • Обжарить их в растительном масле до золотистого цвета
